# gently-cc

Every Claude Code session starts cold. No memory of the last session. No coordination between terminals. Token budgets evaporate on verbose prompts. `gently-cc` is a Claude Code substrate that fixes all three: CODIE compresses every prompt before inference, Alexandria + BARF foam carries memory across sessions without embeddings, and a deterministic cold-process detector catches scope drift and duplicate terminals before they waste context.

---

## Install

```bash
npm install -g gently-cc
gently-cc install
```

`install` does four things: writes `~/.gently/agent-identity.json`, wires hooks into `~/.claude/settings.json`, downloads or builds the three Rust binaries (`codie`, `barf`, `codec`), and registers the session GRIM hash.

## CODIE — Compressed Operational Dense Instruction Encoding

44 keywords. Every prompt is compressed by the `codie` binary before it reaches inference. The compressed form is canonical — it is what Claude sees, not the original.

**Core 12 semantic keywords:**

```
pug   ρ  Entry point       bark  β  Fetch/get
spin  ς  Loop/iterate      cali  κ  Define function
elf   ε  Bind variable     turk  τ  Incomplete
fence φ  Constraint/NOT    pin   π  Exact spec
bone  Β  Immutable         blob  Λ  Flexible
biz   μ  Goal/output       anchor ∆  Checkpoint
```

Plus: 6 logic gates · 10 control flow · 2 boolean · 5 geometric · 5 dimensional · 4 meta/generation.

**Compression example:**

```
Human:  "Fetch the user from the database. If not found, return an error.
         Otherwise return a session token."

CODIE:  ρAUTH⟨βuser←@db/users⟨⁇¬found→⊥⟩μ→token⟩

Tokens: Original: 28 · Compressed: 6 · Saved: 78.6%
```

Natural language compresses 33–45%. Structured CODIE expressions compress 80–94.7%.

**Benchmark locally:**

```bash
cargo build --release -p codie
./target/release/codie bench "your message here"
```

## Cold-Process Labels

Deterministic anomaly detection. No LLM. No network. First match wins.

| Label | Name | Fires when |
|-------|------|------------|
| 🔴◉-S | STALE | Same prompt hash seen in last 8 turns |
| 🔴◉-X | EXPIRED | Prompt references a tombstoned CID |
| 🟡◉-T | THIN | Prompt under 12 tokens — likely accidental |
| 🟡◉-D | DRIFT | Scope ID changed mid-session |
| 🟡◉-P | PATH | Working directory changed but scope ID held |
| 🟡◉-C | COLD | Lexical continuity with recent history < 20% |
| 🟡◉-A | AGENCY | Operator-frame markers present (ambiguous intent) |
| 🟢● | OK | All checks passed |

All anomalies are written to `~/.gently/anomalies.jsonl`. Multi-terminal coordination: when the same prompt appears in two terminals, `🟡◉-D` fires and Greg names which terminal should drop — it does not decide for you.

## gently-codegen — CODIE Expressions to Rust

CODIE symbol expressions compile to deterministic Rust source. Same input, same output, every time. Determinism verified 2026-04-27.

**Generate a CRUD module:**

```bash
gently-cc gen --template crud User
```

**Output (`gently-codegen/examples/crud_user.rs`):**

```rust

// output_cid: sha256:8a2c517dd9b805aee39ecbd670fb8abddd35f1ac942aea737df311f929845519
```

## Content-Addressed Project DAG

The entire project state is one string. `project.codex.json` + `plan.dag.json` form a content-addressed DAG where every file has a sha256 CID. The root CID covers the whole tree.

```
root_cid: sha256:741b4b4fae9baee4e6946655999fa997539fab5330a8e03c302a9de9f914512c
```

**Compute the DAG from PLAN.md:**

```bash
gently-cc plan
# Writes plan.dag.json with node CIDs and dependency edges
```

**Materialize and verify all files:**

```bash
gently-cc materialize --verify
# Recomputes every output_cid and checks against codex
```

**Clone a project from a single string:**

```bash
gently-cc clone sha256:741b4b4fae9baee4e6946655999fa997539fab5330a8e03c302a9de9f914512c
# Materializes all files in project.codex.json to the current directory
```

**Phase 2 (pending):** anchor the root CID on Sui so anyone can verify the project state on-chain:

```bash
gently-cc anchor sha256:741b4b4f...
```

## Architecture

Five primitives compose the substrate:

| Primitive | What it does |
|-----------|-------------|
| **GRIM** | SHA-256-addressed function dispatch. Every capability is a hash. Unknown hash → `None` → stop. Injection is impossible by construction. |
| **CODIE** | 44-keyword compression language. 94.7% token reduction. The compressed form is canonical — it is what the model sees. |
| **Alexandria** | 5W tesseract knowledge graph — 384 dimensions across 8 faces (ACTUAL / ELIMINATED / POTENTIAL / TEMPORAL / OBSERVER / CONTEXT / METHOD / PURPOSE). Session state survives compaction here. |
| **BS-Artisan** | Toroidal foam storage. BARF retrieval via XOR-distance. No embeddings, no cosine similarity. `BARF(hash) → rank tori by XOR(hash, torus_id) / 256 → top-N`. |
| **Sui Move** | Linear-type ownership. SYNTH token has `no copy, no drop`. Capabilities locked at NFT mint — no runtime expansion. |

Agent identity (`Greg`, theme `Crimson`, project_id `a3f91bc2e47d58f0`) is deterministic. Theme is `SHA-256(project_id) % 16`. Identity is written at install and does not change unless the project is renamed.

```bash
cargo build --release -p codie          # compression
cargo build --release -p bs-artisan     # BARF foam retrieval
cargo build --release -p gently-codec   # hierarchical pin grid
```

Or all at once:

```bash
npm run stack:build
```

**Run tests:**

```bash
node --test src/__tests__/
```

**Verify skill hashes and hook wiring:**

```bash
gently-cc verify
```

```bash
gently-cc recover
```
